# OpenMarquee 0.3.0

This release adds production-focused administration and Windows distribution.

## Highlights

- Administrator-only software shutdown that stops live sharing and safely disconnects players without shutting down the computer.
- Automatic administrator logout after 10 minutes of inactivity, enforced by both the browser and server session.
- Media renaming plus folder rename and delete controls. Deleting a folder preserves its media in Unfiled.
- Expanded in-app FAQ covering Windows installation, live sharing, media organization, session security, and shutdown behavior.
- Windows setup with modern installation, uninstall support, OpenMarquee Desktop and Start Menu shortcuts, and a portable ZIP edition.
- Packaged application data remains under `%LOCALAPPDATA%\OpenMarquee` so upgrades do not remove content or settings.

## Downloads

- `OpenMarquee-Setup-v0.3.0.exe`: recommended Windows installer.
- `OpenMarquee-v0.3.0-portable.zip`: extract and run `OpenMarquee.exe` without installation.
